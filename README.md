# supermarket
# 超市管理系统

一个基于 Spring Boot + Vue 的前后端分离的超市管理系统。

## 一、项目介绍

本系统实现了超市日常运营的核心功能，包括商品管理、供应商管理、订单管理等。

## 二、功能特性

- 用户登录/注册
- 商品增删改查
- 库存预警
- 订单详情管理
- 供应商管理

## 三、技术栈

后端：
- Spring Boot 3.2.0
- MyBatis Plus
- MySQL 8.0.40
- Redis

前端：
- Vue 3
- Element Plus
- Axios
- Vite

## 四、快速开始

### 环境要求
- JDK 17+
- Node.js 16+
- MySQL 8.0.40+
- Maven 3.9.8+

### 后端启动
启动SpringbootApplication

### 后端启动
终端: cd supermarket/vue
#### npm install
#### npm run dev
